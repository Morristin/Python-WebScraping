## Python WebScraping

> 此仓库是所有者 2025-2026 学年第二学期 课程「Python 程序设计」的课程设计。

你目前正在访问的是本仓库的 `interact` 分支。
**该分支为进行现场操作进行了特殊额外优化，你必须要具有一定的 Python 能力以使用此程序**。

参见 `main` 分支的 README 文档获取更多信息：https://github.com/Morristin/Python-WebScraping/tree/main 。

### Installation

你必须具有相关 git 操作能力自行切换到此分支并安装所需依赖。之后，运行 `main.py` 进入交互式命令行。

```shell
python3 main.py --debug
```

此分支下，只允许使用 `debug` 模式运行程序。否则程序将发出提示并直接终止。
